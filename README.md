# Bronz

Dolgozó nyilvántartás

REST API kliens JavaScript nyelven

| végpont     |  metódus | auth | CRUD  | leírás |
|-------------|----------|------|-------|---------|
| employees   | GET      | nem  | read  | összes dolgozó lekérése  |
| employees   | POST     | nem  | create | új dolgozó hozzáadása  |
| employees/{id} |  PUT  | nem  | update | dolgozó adatainek frissítése |
| employees/{id} | DELETE | nem | delete |   dolgozó törlése |
